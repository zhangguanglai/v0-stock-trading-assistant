# 策略大师 - 个人股票投资助手

Stock Investment Strategy Master - A Professional Personal Investment Assistant

## 📋 项目简介

策略大师是一个专业级的个人股票投资助手应用，帮助散户投资者建立**规则化交易体系**，实现**纪律化操作**。该系统强调"规则执行"与"纪律训练"，帮助投资者克服情绪化操作，建立可持续的投资体系。

## 🎯 核心功能

### 1️⃣ 策略配置中心
- **自定义策略** - 灵活配置选股规则、买入条件、卖出规则、资金管理
- **预设模板** - 4个专业策略模板：波段交易Pro、价值投资系统、趋势跟踪系统、成长股投资系统
- **规则检查** - 买入/卖出时弹出规则检查，确保每笔交易符合系统规则
- **纪律训练** - 完整的规则执行记录和纪律遵守率统计

### 2️⃣ 智能股票池
- **策略筛选** - 基于策略自动筛选符合条件的股票
- **多策略切换** - 当激活≥2个策略时，可快速切换查看不同策略的筛选结果
- **股票自选** - 标记重点关注股票，持续跟踪
- **手动补充** - 支持手动添加自选股票

### 3️⃣ 买入信号灯 & 仓位计算器
- **信号检测** - 实时检测技术面买入信号（MA金叉、MACD、RSI等）
- **仓位计算** - 基于凯利公式计算最优仓位
- **风险参数** - 计算止损点位、头寸大小、分批建议
- **资金管理** - 单股仓位限制、总仓位限制、单笔风险限制

### 4️⃣ 止盈止损哨兵 (持仓管理)
- **实时监控** - 监控活跃持仓的价格变动
- **智能止损** - 支持固定止损、移动止损、时间止损
- **止盈管理** - 分批止盈、部分止利设置
- **风险提醒** - 超过最大亏损时主动提醒

### 5️⃣ 风险透视仪
- **组合分析** - 投资组合风险评估和得分
- **行业集中度** - 防止行业过度集中风险
- **资金分配** - 已用资金/可用资金/现金持仓比例
- **压力测试** - 模拟极端行情下的组合表现
- **波动率分析** - 个股和组合的波动率和夏普比率

### 6️⃣ 交易日志与复盘
- **完整记录** - 每笔交易的时间、价格、原因、情绪状态
- **系统分析** - 胜率、平均收益、平均亏损、盈亏比、夏普比率
- **交易复盘** - 记录经验教训和改进方向
- **不适期识别** - 连续亏损时标注"不适期"并给予鼓励

## 🛠 技术栈

| 层级 | 技术 |

## 💾 数据库架构

### 表结构
- `user_profiles` - 用户配置 (资金、风险参数)
- `strategies` - 交易策略配置
- `stock_pools` - 股票池管理
- `stock_pool_items` - 股票池中的股票
- `positions` - 持仓记录
- `trades` - 交易记录
- `trade_reviews` - 交易复盘
- `alerts` - 信号提醒

### 安全特性
- ✅ 行级安全(RLS)策略 - 用户只能访问自己的数据
- ✅ 级联删除 - 用户删除时自动清理相关数据
- ✅ 自动触发器 - 新用户注册时自动创建配置

## 🚀 快速开始

### 1. 环境配置

创建 `.env.local` 文件：

```bash
NEXT_PUBLIC_SUPABASE_URL=your_supabase_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_supabase_anon_key
```

### 2. 本地开发

```bash
# 安装依赖
npm install

# 启动开发服务器 (http://localhost:3000)
npm run dev

# 构建生产版本
npm run build

# 启动生产服务器
npm start
```

### 3. 部署到Vercel

```bash
# 推送到GitHub
git push

# Vercel会自动部署
# 访问: https://your-project.vercel.app
```

### 已实现的优化

## 📚 使用流程

### 新用户流程
1. 注册账号 (邮箱/密码)
2. 策略配置中心 → 从模板创建策略
3. 智能股票池 → 查看筛选结果
4. 加入自选 → 关注重点股票
5. 仓位计算器 → 计算买入参数
6. 买入时执行规则检查

### 日常运营流程
1. 查看"投资总览" → 了解组合状态
2. 检查"持仓管理" → 监控止损止盈
3. 接收"信号提醒" → 及时响应市场机会
4. 完成交易后 → 记录在"交易日志"
5. 周末复盘 → 在"交易复盘"中总结经验
