## 🔧 部署说明

### 快速开始
```bash
# 1. 克隆项目
git clone https://github.com/yourusername/stock-strategy-master.git
cd stock-strategy-master

# 2. 安装依赖
npm install

# 3. 设置环境变量
cp .env.example .env.local
# 编辑.env.local，填入Supabase配置

# 4. 运行本地开发
npm run dev

# 5. 访问应用
# 打开 http://localhost:3000
```

### 部署到Vercel
```bash
# 方式1: 通过Vercel Dashboard
# 1. 访问 https://vercel.com
# 2. 导入GitHub仓库
# 3. 添加环境变量
# 4. 点击Deploy

# 方式2: 通过CLI
vercel
# 按提示输入配置信息
```

### 数据库初始化
- 创建Supabase项目
- 运行DEPLOYMENT.md中的初始化脚本
- 配置RLS策略（已在脚本中）

## 🎓 使用流程示例

### 第一次使用
1. 注册账号
2. 从模板创建策略 → "波段交易系统Pro"
3. 进入智能股票池，查看符合条件的股票
4. 选择感兴趣的股票加入自选
5. 使用仓位计算器计算最优头寸
6. 执行买入，系统会检查是否符合规则
7. 在持仓管理中设置止盈止损
8. 每周在交易复盘中总结经验

### 日常操作流程
1. 每天查看"投资总览"了解组合状态
2. 检查"持仓管理"中是否有提醒
3. 在"智能股票池"中发现新机会
4. 如有交易，完成后在"交易日志"中记录
5. 周末进行"交易复盘"
